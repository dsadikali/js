A) Description - top

The giga image viewer jQuery plugin allows you to easily replace <div> tags with animated image viewers with hotspots.

Viewer used for displaying very high-resolution JPG.
It is a completely packaged to present high resolution images on the web.
Package contains command line program (for Windows only) and php script, for cutting images into pieces
(max tested image had size 9000x9000 px).

Viewer displays the given high-resolution JPG inside the user-defined viewport area.
Viewer allows to control the position and zoom of the high-resolution JPG inside the viewport.
Viewer controls the sliding and zoom of the high-resolution JPG so that the viewport area will be filled completely.

Through using this plugin you can also protect high resolution images.

Features:

  • Touch support for mobile devices.
  • Mouse or Touch drag.
  • Mouse wheel to zoom / unzoom.
  • Mouse click or Touch move to center viewport.
  • Possible viewport resize on runtime.
  • Package contains a program for cutting image.
  • Package contains a example for building thumbnail gallery.
  • Package contains a example for building lightbox gallery (ColorBox Gallery).
  • Cross-browser compatible – Image Viewer is compatible with IE, CHROME , FIREFOX, OPERA , SAFARI.
  • Very easy installation on your html page.
  • Very smooth animations.
  • Fully scalable.
  • Displays the map palette to quickly change the view using a thumbnails.
  • Placing hotspots on images.
  • Addition popup windows on the hotspot.

Viewer functionality:

  • set start coordinates and scale
  • set animation duration
  • set drag inertia
  • set the navigation bar (enabled / disabled)
  • set the navigation bar autohide
  • set the navigation bar position (acceptable values: TL, T, TR, L, R, BL, B, BR)
  • set visibility of the buttons in the navigation bar
  • set the the map palette (enabled / disabled)
  • set the the map palette position (acceptable values: TL, T, TR, L, R, BL, B, BR)
  • reading current image position and scale
  • dispatch event ‘givChange’ with information whether it is possible to move and zoom the display object

C) Plugin methods - top

Method name Example use Description
setPosition $('#myDiv').lhpGigaImgViewer('setPosition', x, y, scale, noAnim) Sets the position and size of the displayed object. The scale parameter is optional - if null, the size remains unchanged.

@param {integer} x Coordinate x in px, in the display object frame of reference, which will be moved to the center of the viewport (if it is possible).
@param {integer} y Coordinate y in px, in the display object frame of reference, which will be moved to the center of the viewport (if it is possible).
@param {number} scale The size to which the display object will be scaled (if it is possible); optional.
@param {boolean} noAnim Animations disabled; optional, default false.
@return {Object} Returns jQuery object.
moveUp $('#myDiv').lhpGigaImgViewer('moveUp') Initializes the movement of the display object to the top, to the boundary of the viewport or untill the moveStop method is called.

@return {Object} Returns jQuery object.
moveDown $('#myDiv').lhpGigaImgViewer('moveDown') Initializes the movement of the display object to the bottom, to the boundary of the viewport or untill the moveStop method is called.

@return {Object} Returns jQuery object.
moveLeft $('#myDiv').lhpGigaImgViewer('moveLeft') Initializes the movement of the display object to the left, to the boundary of the viewport or untill the moveStop method is called.

@return {Object} Returns jQuery object.
moveRight $('#myDiv').lhpGigaImgViewer('moveRight') Initializes the movement of the display object to the right, to the boundary of the viewport or untill the moveStop method is called.

@return {Object} Returns jQuery object.
moveStop $('#myDiv').lhpGigaImgViewer('moveStop') Stops the movement of the display object.

@return {Object} Returns jQuery object.
zoom $('#myDiv').lhpGigaImgViewer('zoom') Initializes the zooming of the display object up to 100% or untill the zoomStop method is called.

@return {Object} Returns jQuery object.
unzoom $('#myDiv').lhpGigaImgViewer('unzoom') Initializes the unzooming of the display object up to the viewport's size or untill the zoomStop method is called.

@return {Object} Returns jQuery object.
zoomStop $('#myDiv').lhpGigaImgViewer('zoomStop') Stops the zooming/unzooming of the display object.

@return {Object} Returns jQuery object.
fitToViewport $('#myDiv').lhpGigaImgViewer('fitToViewport') Fits the display obejct's size to the viewport size and moves the object to the center of the viewport.

@return {Object} Returns jQuery object.
fullSize $('#myDiv').lhpGigaImgViewer('fullSize') Sets the initial size of the display object and moves the object to the center of the viewport.

@return {Object} Returns jQuery object.
adaptsToContainer $('#myDiv').lhpGigaImgViewer('adaptsToContainer') Control the correct position and size of the object displayed inside the viewport. IMPORTANT - Method should be called after each the outer div resize.

@return {Object} Returns jQuery object.
getCurrentState $('#myDiv').lhpGigaImgViewer('getCurrentState') Returns object with control values. The object is identical as the object returned by the event 'givChange', read more - Plugin event.

@return {Object} Returns object with control values.
destroy $('#myDiv').lhpGigaImgViewer('destroy') Destructor. Removes the viewer from the page. Restores the original appearance and functionality of the outer <div> element. Allows to efficiently clean the memory.

@return {Object} Returns jQuery object.

D) Plugin event - top

givChange

This plugin dispatch event 'givChange' after modifying the control value. Control values contains information:

  • current display object scale
  • current display object position
  • scaling in progress
  • whether it is possible to move the display object up, down, left or right
  • whether it is possible to zoom and unzoom
  • proportion viewport to display object

This event has the following properties:
scale - number, current scale display object (0...1).
xPosInCenter - number, current x position display object in viewport center (px).
yPosInCenter - number, current y position display object in viewport center (px).
allowUp - true, if the movement in the "up" direction is possible, false otherwise.
allowDown - true, if the movement in the "down" direction is possible, false otherwise.
allowLeft - true, if the movement in the "left" direction is possible, false otherwise.
allowRight - true, if the movement in the "right" direction is possible, false otherwise.
allowZoom - true, if zooming is possible, false otherwise.
allowUnzoom - true, if unzooming is possible, false otherwise.
hPropViewpContent - number, proportion (vertical) viewport to display object.
wPropViewpContent - number, proportion (horizontal) viewport to display object.
isScaled - true, when scaling in progress, false otherwise.

EXAMPLE USE: 

$('#myDiv').bind("givChange", function(e) {
	alert(e.scale + ', ' + e.xPosInCenter + ', ' + e.allowLeft + ', ' + e.allowRight);
	alert(e.allowDown + ', ' + e.allowUp + ', ' + e.yPosInCenter);
	alert(e.allowZoom + ', ' + e.allowUnzoom);
});

E) Basic usage. - top

Preparing (cutting) of the source image

The package contains a program 'imageslicer.exe' used to prepare images. A program for run requires Windows operating system. Imageslicer cuts the main image, creates preview image (visible when loading main image) and creates image for map palette (map thumb - quick view). Largest tested image had size 9000x9000 px.

  • STEP 1

    Create new directory (example c:\giv) on your local machine with Windows operating system.
    Copy files to a new directory:
    - imageslicer.exe from download package (imageslicer/imageslicer.exe)
    - and your source image to cutting (infile.jpg)
    Create new directory (example c:\giv\output) for the output images.

  • STEP 2

    Run command line more info.
    Go to 'c:\giv' directory.

  • STEP 3 - Run imageslicer.exe

    Running program requires two parameters (and/or two optional):
    - first parameter is a path to the source image, in this case 'c:\giv\infile.jpg' or 'infile.jpg'
    - second parameter is a path to the directory for the output images, in this case 'c:\giv\output\' or 'output\'
    - third parameter (optional) is size of the preview image, default 512px
    - fourth parameter (optional) is size of the map thumb image, default 100px

    You can run a program without parameters to displays the quick help:

    Running the program with the required parameters:


    After cutting and having created the rest of the image files (operation may take several minutes for large images),
    imageslicer displays the size of the main image, this size you must enter in plugin settings. 

    var settings = {
	...
	'mainImgWidth' : 5000,
	'mainImgHeight' : 5000,
	...
};

  • STEP 4

    Upload the contents of the c:\giv\output\ directory to a server directory (example 'hubbleImgSrc/') and set the path in the plugin settings. 

    var settings = {
	...
	'imgDir' : 'hubbleImgSrc/',
	...
};

Installation & Setup

  • STEP 1 - INCLUDE THE CSS AND SCRIPT

    1. Embed the CSS files in the <head> section of your HTML document, set the src="css-path/" to your css file directory.
    The custom-theme is required for navigation bar.

    <link rel="stylesheet" type="text/css" href="css/custom-theme/jquery-ui-1.10.3.custom.css" />
<link rel="stylesheet" type="text/css" href="css/lhp_giv.css" />

    2. Copy 'custom-theme' and 'lhp_giv' directory from css to your css file directory.

    3. Embed the JavaScript files in the <head> section of your HTML document, set the src="js-path/" to your JavaScript file directory. The jQuery Easing Plugin is required for easing effects. The jQuery Mouse Wheel Plugin is required for mouse wheel interaction.

    <script type="text/javascript" src="js/jquery-1.10.1.min.js"></script>
<script type="text/javascript" src="js/jquery.easing.1.3.js"></script>
<script type="text/javascript" src="js/jquery.mousewheel.min.js"></script>

G) Using hotspots - top

  • STEP 1 - create empty div, set the id and set display none
    This is hotspots container. 
    <div id="MyHotspots" style="display:none;"></div>
  • STEP 2 - create hotspot div, set the class lhp_giv_hotspot and put this div inside hotspots container.
    This is a single hotspot container. 
    <div id="MyHotspots" style="display:none;">
	<div class="lhp_giv_hotspot"></div>
</div>
  • STEP 3 - set position of the hotspot: data-x, data-y
    and set scale above which hotspot is shown: data-visible-scale, (accepted: 0...1).
    <div id="MyHotspots" style="display:none;">
	<div class="lhp_giv_hotspot" data-x="720" data-y="500" data-visible-scale="0"></div>
</div>
  • STEP 4 - add a marker div, set the class lhp_giv_marker
    and specify marker location relative to the hotspot location (accepted: TL, T, TR, L, C, R, BL, B, BR).
    This is a marker container. 
    <div id="MyHotspots" style="display:none;">
	<div class="lhp_giv_hotspot" data-x="720" data-y="500" data-visible-scale="0">
		<div class="lhp_giv_marker pos-BL">MARKER #1</div>
	</div>
</div>
  • STEP 5 (optional) - add a popup div, set the class lhp_giv_popup
    and specify popup location relative to the marker location (accepted: TL, T, TR, L, R, BL, B, BR).
    This is a popup container. 
    <div id="MyHotspots" style="display:none;">
	<div class="lhp_giv_hotspot" data-x="720" data-y="500" data-visible-scale="0">
		<div class="lhp_giv_marker pos-BL">MARKER #1</div>
		<div class="lhp_giv_popup pos-L">POPUP CONTENT</div>
	</div>
</div>
  • STEP 6 add another hotspot.
    Repeat steps 2-5.

  • STEP 7 activate the plugin with hotspots.
    $('#myDiv').lhpGigaImgViewer(settings, 'MyHotspots');

H) Navigation bar - theme - top

The gig image viewer jQuery plugin uses Themeroller to determine appearance navigation bar.

If you want change theme:

  1. go to http://jqueryui.com/themeroller/
  2. set your settings and click 'Download theme', you'll be directed to the Download Builder and your custom theme will be auto-selected in the Theme dropdown menu
  3. click 'Download' button and save zip file
  4. unpack zip file and copy the contents of css folder to your css folder
  5. change this link in your head
    <link rel="stylesheet" type="text/css" href="css/custom-theme/jquery-ui-1.10.3.custom.css" />
    for example, if you downloaded the redmond theme would be:
    <link rel="stylesheet" type="text/css" href="css/redmond/jquery-ui-1.10.3.custom.css" />
